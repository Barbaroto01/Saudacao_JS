# Saudação Interativa

Aplicação web simples em JavaScript que recebe o nome do usuário e exibe uma mensagem de saudação personalizada.

## Tecnologias
- HTML
- CSS
- JavaScript

## Funcionalidades
- Entrada de nome pelo usuário
- Exibição de saudação personalizada
- Interface simples e centralizada

## Como executar

Abra o projeto diretamente pelo navegador através do GitHub Pages:

https://barbaroto01.github.io/Saudacao_JS/

## Aprendizados
Projeto criado para praticar:
- Entrada de dados do usuário
- Manipulação de texto
- Funções em JavaScript
- Integração entre HTML, CSS e JavaScript
